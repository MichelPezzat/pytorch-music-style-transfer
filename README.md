# Many-to-Many Symbolic Multi-track Music Genre Transfer
This the pytorch implementation of StarGAN-based model to realize music style transfer between different musical domains with a single generator.

# Dependencies

* Python 3.6+
* pytorch 1.0
* librosa
* tensorboardX
* scikit-learn
* numpy 1.14.2
* pretty_midi 0.2.8
* pypianoroll 0.1.3
